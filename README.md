# GreenShop — Django интернет-магазин

Полнофункциональное e-commerce приложение на Django 4.2 с каталогом товаров, корзиной, оформлением заказа, личным кабинетом и настраиваемой админкой. Интерфейс выполнен в зеленой палитре (`#28a745`, `#155724`, `#20c997`) на Bootstrap 5 с кастомными стилями и адаптивной версткой.

## Возможности
- Категории, карточки товаров, галерея изображений, поиск, фильтры и пагинация.
- Сессионная корзина с Ajax‑обновлением, изменением количества и подсчетом суммы.
- Оформление заказа, история заказов, личный кабинет и профиль пользователя.
- Регистрация/авторизация, пользовательские профили создаются автоматически.
- Админ-панель с управлением товарами/категориями и статистикой продаж.
- Команда `seed_store` для наполнения базы демо-данными.

## Стек
- Django 4.2, SQLite3
- Bootstrap 5, Custom CSS, JavaScript (Fetch API)
- Pillow для работы с изображениями

## Быстрый старт
```bash
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser  # по желанию
python manage.py load_catalog_data --flush  # демо-каталог
python manage.py runserver
```

После запуска:
- `http://127.0.0.1:8000/` — витрина и каталог
- `http://127.0.0.1:8000/admin/` — админка (после создания суперпользователя)

Статические файлы уже настроены (`static/`), кастомные стили — в `static/css/custom.css`, сценарий динамики корзины — `static/js/cart.js`, шаблоны — в `templates/`. Для обновления демонстрационных категорий и товаров повторно выполните `python manage.py load_catalog_data --flush`.
